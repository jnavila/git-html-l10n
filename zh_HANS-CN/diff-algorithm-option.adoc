`--diff-algorithm=(patience|minimal|histogram|myers)`::
	选择差异算法。有如下可选项：
+
--
   `default`;;
   `myers`;;
	基本的贪婪差异算法。当前是默认设置。
   `minimal`;;
	花费额外的时间以确保生成尽可能小的差异。
   `patience`;;
	使用 "patience diff" 算法时产生的补丁。
   `histogram`;;
	该算法将耐心算法扩展为 “支持低发生率的常见元素”。
--
+
例如，如果你将 `diff.algorithm` 变量配置为非默认值，但希望使用默认值，那么你必须使用 `--diff-algorithm=default` 选项。

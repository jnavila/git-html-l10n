`clone.defaultRemoteName`::
	克隆仓库时创建的远程仓库的名称。默认为 `--origin`。
ifdef::git-clone[]
	可在命令行传入 `--origin` 参数
	覆盖该配置。
endif::[]
ifndef::git-clone[]
	它可以通过向 linkgit:git-clone[1] 
	传递 `--origin` 命令行选项来覆盖。
endif::[]

`clone.rejectShallow`::
	如果是浅层仓库，则拒绝克隆；可以通过在命令行中传递 `--reject-shallow` 选项来重写。
ifndef::git-clone[]
	参见 linkgit:git-clone[1].
endif::[]

`clone.filterSubmodules`::
	如果提供了部分克隆过滤器（参见 linkgit:git-rev-list[1] 中的 `--filter` 选项），并且如果使用了 `--recurse-submodules` 选项，也会将过滤器应用于子模块。

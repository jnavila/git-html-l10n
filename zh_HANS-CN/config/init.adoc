`init.templateDir`::
	指定将要复制模板的目录。{see-git-init}
`init.defaultBranch`::
	允许覆盖默认分支名称，例如在初始化新仓库时。
`init.defaultObjectFormat`::
	允许覆写新仓库的对象格式。参见 linkgit:git-init[1] 中的 `--object-format=`。并且，命令行选项和环境变量 `GIT_DEFAULT_HASH` 都优先于该配置。
`init.defaultRefFormat`::
	允许覆写新仓库的引用存储格式。参见 linkgit:git-init[1] 中的 `--ref-format=`。命令行选项和环境变量 `GIT_DEFAULT_REF_FORMAT` 都优先于该配置。

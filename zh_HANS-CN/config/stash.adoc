`stash.index`::
	如果设为 true，`git stash apply` 和 `git stash pop` 的行为将如同提供了 `--index`。默认值为 false。
ifndef::git-stash[]
参见 linkgit:git-stash[1] 中的说明。
+
这也会影响通过 `--autostash` 间接调用 linkgit:git-stash[1] 的情况，例如来自 linkgit:git-merge[1]、linkgit:git-rebase[1] 和 linkgit:git-pull[1] 的调用。
endif::git-stash[]

`stash.showIncludeUntracked`::
	如果设为 true，则 `git stash show` 命令会显示 stash 条目中的未跟踪文件。默认值为 false。{see-show}

`stash.showPatch`::
	如果设为 true，则不带选项的 `git stash show` 命令会以 patch 形式显示 stash 条目。默认值为 false。{see-show}

`stash.showStat`::
	如果设为 true，则不带选项的 `git stash show` 命令会显示 stash 条目的 diffstat。默认值为 true。{see-show}

git-verify-commit(1)
====================

名称
--
git-verify-commit - 检查提交的 GPG 签名

概述
--
[synopsis]
git verify-commit [-v | --verbose] [--raw] <commit>...

描述
--
Validates the GPG signature created by `git commit -S` on the commit objects given on the command line.

选项
--
`--raw`::
	将原始 gpg 状态输出打印到标准错误，而不是正常的人类可读输出。

`-v`::
`--verbose`::
	在验证之前打印提交对象的内容。

git-citool(1)
=============

名称
--
git-citool —— 替代 git-commit 的图形化工具

概述
--
[verse]
'git citool'

描述
--
一个基于 Tcl/Tk 的图形界面，用于查看修改过的文件，将其放入索引，输入提交信息，并将新的提交记录到当前分支。 这个界面是对互动性较差的 'git commit' 程序的一种替代。

'git citool' is actually a standard alias for `git gui citool`. See linkgit:git-gui[1] for more details.

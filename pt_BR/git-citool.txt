git-citool(1)
=============

NOME
----
git-citool - Alternativa gráfica para git-commit

SINOPSE
-------
[verse]
'git citool'

DESCRIÇÃO
---------
Uma interface gráfica baseada em Tcl/Tk para revisar arquivos modificados,
colocá-los no índice, inserir uma mensagem de confirmação e registrar o novo
commit no branch atual.  Essa interface é uma alternativa ao programa menos
interativo 'git commit'.

'git citool' é na verdade um pseudônimo padrão para `git gui citool`. Veja
linkgit:git-gui[1] para mais detalhes.

GIT
---
Parte do linkgit:git[1] suite

git-init-db(1)

NOME
----
git-init-db - Cria um repositório Git vazio


SINOPSE
-------
[verse]
'git init-db' [-q | --quiet] [--bare] [--template=<diretório_modelo>] [--separate-git-dir <git dir>] [--shared[=<permissões>]]


DESCRIÇÃO
---------

Este é um sinônimo para linkgit:git-init[1].  Para referências deste
comando, favor consultar a documentação.

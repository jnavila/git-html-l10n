`merge.branchdesc`::
	Além dos nomes dos ramos, preencha a mensagem do registro log com o texto de descrição do ramo associado a eles. A predefinição é 'false'.

`merge.log`::
	Além dos nomes dos ramos, preencha a mensagem do registro log com, no máximo, a quantidae especificado de descrições de uma linha dos commits reais que estão sendo mesclados. A predefinição é "false", e "true" é um sinônimo para 20.

`merge.suppressDest`::
	By adding a glob that matches the names of integration branches to this multi-valued configuration variable, the default merge message computed for merges into these integration branches will omit "into _<branch-name>_" from its title.
+
Um elemento com um valor vazio pode ser usado para limpar a lista de globs acumulados de entradas de configuração anteriores. Quando não há uma variável `merge.suppressDest` definida, o valor predefinido `master` é usado para ser compatível com as versões anteriores.

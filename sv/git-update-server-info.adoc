git-update-server-info(1)
=========================

NAMN
----
git-update-server-info - Uppdatera reserv-info filen för att hjälpa dumma servrar


SYNOPSIS
--------
[verse]
'git update-server-info' [-f | --force]

BESKRIVNING
-----------
En dum-server som inte genererar paket i realtid måste ha vissa reserv-informations-filer i katalogerna $GIT_DIR/info och $GIT_OBJECT_DIRECTORY/info för att hjälpa klienter att upptäcka vilka referenser och paket servern har. Detta kommando genererar sådana reserv-informations-filer.

ALTERNATIV
----------
-f::
--force::
	Uppdatera informationsfilerna från grunden.

UTMATNING
---------

För närvarande uppdaterar kommandot följande filer. Se linkgit:gitrepository-layout[5] för en beskrivning av vad de är till för:

* objects/info/packs

* info/refs

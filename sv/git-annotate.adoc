git-annotate(1)
===============

NAMN
----
git-annotate - Annotera filrader med inchecknings-information

SYNOPSIS
--------
[verse]
git annotate [<flaggor>] [<rev-flaggor>] [<rev>] [--] <fil>

BESKRIVNING
-----------
Annoterar varje rad i den givna filen med information från den incheckning som introducerade raden. Annoterar valfritt från en given revision.

Den enda skillnaden mellan detta kommando och linkgit:git-blame[1] är att de använder något olika utdataformat, och detta kommando finns endast för bakåtkompatibilitet för att stödja befintliga skript och ge ett mer välbekant kommandonamn för personer som kommer från andra SCM-system.

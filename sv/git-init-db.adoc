git-init-db(1)
==============

NAMN
----
git-init-db - Skapar ett tomt Git-repo


SYNOPSIS
--------
[verse]
'git init-db' [-q | --quiet] [--bare] [--template=<mallkatalog>] [--separate-git-dir <git-kat>] [--shared[=<behörigheter>]]


BESKRIVNING
-----------

Detta är en synonym för linkgit:git-init[1]. Se dokumentationen för det kommandot.

git-init-db(1)
==============

NAMN
----
git-init-db - Skapar ett tomt Git-kodförråd


SYNOPSIS
--------
[verse]
'git init-db' [-q | --quiet] [--bare] [--template=<mallkatalog>] [--separate-git-dir <git-kat>] [--shared[=<behörigheter>]]


BESKRIVNING
-----------

Det här är en synonym för linkgit:git-init[1]. Se dokumentationen för det kommandot.

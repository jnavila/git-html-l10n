git-unpack-file(1)
==================

NAMN
----
git-unpack-file - Skapar en temporär fil med innehållet i en blob



SYNOPSIS
--------
[verse]
'git unpack-file' <blob>

BESKRIVNING
-----------
Skapar en fil som innehåller innehållet i den blob som anges av
sha1. Den returnerar namnet på den temporära filen i följande
 format:
	.merge_file_XXXXX

ALTERNATIV
----------
<blob>::
	Måste vara ett blob-id

GIT
---
En del av linkgit:git[1]-sviten

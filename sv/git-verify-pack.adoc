git-verify-pack(1)
==================

NAMN
----
git-verify-pack - Validerar packade Git-packfiler


SYNOPSIS
--------
[synopsis]
git verify-pack [-v | --verbose] [-s | --stat-only] [--] <paket>.idx...


BESKRIVNING
-----------
Läs varje idx-fil för den packade Git-packfilen som anges på kommandoraden och verifiera idx-filen och motsvarande packfil.

ALTERNATIV
----------
`-v`::
`--verbose`::
	Efter att ha verifierat paketet, visa listan över objekt som finns i paketet och ett histogram över deltakedjelängden.

`-s`::
`--stat-only`::
	Verifiera inte paketets innehåll; visa endast histogrammet för deltakedjelängden. Med `--verbose` visas även listan över objekt.

`--`::
	Tolka inte fler argument som alternativ.

UTMATNINGSFORMAT
----------------
När man anger `-v`-alternativet används följande format:

	objektnamn typ storlek storlek-i-paketfil offset-i-paketfil

för objekt som inte är deltakodade i förpackningen, och

	objektnamn typ storlek storlek-i-paketfil offset-i-paketfil djup basobjektnamn

för objekt som är deltakodade.

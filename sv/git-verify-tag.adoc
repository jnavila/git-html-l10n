git-verify-tag(1)
=================

NAMN
----
git-verify-tag - Kontrollera GPG-signaturen för taggar

SYNOPSIS
--------
[synopsis]
git verify-tag [-v | --verbose] [--format=<format>] [--raw] <tagg>...

BESKRIVNING
-----------
Validerar gpg-signaturen som skapats av `git tag` i taggobjekten som listas på kommandoraden.

ALTERNATIV
----------
`--raw`::
	Skriv ut rå GPG-statusutdata på standardfel i stället för normal människoläsbar utdata.

`-v`::
`--verbose`::
	Skriv ut innehållet i taggobjektet innan du validerar det.

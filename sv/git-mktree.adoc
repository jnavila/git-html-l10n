git-mktree(1)
=============

NAMN
----
git-mktree - Bygg ett trädobjekt från ls-tree-formaterad text


SYNOPSIS
--------
[verse]
'git mktree' [-z] [--missing] [--batch]

BESKRIVNING
-----------
Läser standardindata i icke-rekursivt `ls-tree`-utdataformat och skapar ett trädobjekt. Ordningen på trädposterna normaliseras av mktree, så försortering av indata krävs inte. Objektnamnet på det skapade trädobjektet skrivs till standardutdata.

ALTERNATIV
----------
-z::
	Läs i stället den NUL-avslutade `ls-tree -z`-utdatan.

--missing::
	Tillåt saknade objekt. Standardbeteendet (utan detta alternativ) är att verifiera att varje trädposts hash identifierar ett befintligt objekt. Detta alternativ har ingen effekt på behandlingen av gitlink-poster (även kända som "undermoduler") som alltid tillåts saknas.

--batch::
	Tillåt byggande av mer än ett trädobjekt innan avslutning. Varje träd separeras med en enda tom rad. Den sista nyraden är valfri. Observera - om alternativet `-z` används avslutas raderna med NUL.

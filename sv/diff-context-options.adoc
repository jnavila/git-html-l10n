`-U<n>`::
`--unified=<n>`::
	Generate diffs with _<n>_ lines of context. The number of context lines defaults to `diff.context` or 3 if the configuration variable is unset. (`-U` without `<n>` is silently accepted as a synonym for `-p` due to a historical accident).

`--inter-hunk-context=<n>`::
	Visar sammanhanget mellan olika stycken, upp till det angivna _<antal>_ rader, och sammanfogar därmed stycken som ligger nära varandra. Standardvärdet är `diff.interHunkContext` eller 0 om konfigurationsalternativet inte är inställt.

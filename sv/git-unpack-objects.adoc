git-unpack-objects(1)
=====================

NAMN
----
git-unpack-objects - Packa upp objekt från ett packat arkiv


SYNOPSIS
--------
[verse]
'git unpack-objects' [-n] [-q] [-r] [--strict]


BESKRIVNING
-----------
Läs ett packat arkiv (.pack) från standardindata, expandera objekten inuti och skriv dem till kodförrådet i "löst" format (ett objekt per fil).

Objekt som redan finns i kodförrådet kommer *inte* att packas upp från packfilen. Därför, kommer ingenting att packas upp om du använder det här kommandot på en packfil som finns i mål kodförrådet.

Se linkgit:git-repack[1] för alternativ för att generera nya paket och ersätta befintliga.

ALTERNATIV
----------
-n::
Testkörning. Kontrollera paketfilen utan att faktiskt packa upp objekten.

-q::
	Kommandot visar vanligtvis procentuella framsteg. Denna flagga undertrycker det.

-r::
	När man packar upp en korrupt packfil, dör kommandot vid den första korruptionen. Denna flagga säger åt det att fortsätta och göra sitt bästa för att återställa så många objekt som möjligt.

--strict::
	Skriv inte objekt med trasigt innehåll eller länkar.

--max-input-size=<storlek>::
	Dö, om paketet är större än <storlek>.

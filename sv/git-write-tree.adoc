git-write-tree(1)
=================

NAMN
----
git-write-tree - Skapa ett trädobjekt från det aktuella indexet


SYNOPSIS
--------
[synopsis]
git write-tree [--missing-ok] [--prefix=<prefix>/]

BESKRIVNING
-----------
Skapar ett trädobjekt med hjälp av det aktuella indexet. Namnet på det nya trädobjektet skrivs ut som standardutdata.

Indexet måste vara i ett helt sammanfogat tillstånd.

Konceptuellt sett synkroniserar `git write-tree` det aktuella indexinnehållet till en uppsättning trädfiler. För att det ska matcha vad som faktiskt finns i din katalog just nu måste du ha genomfört en `git update-index`-fas innan du genomförde `git write-tree`.


ALTERNATIV
----------
`--missing-ok`::
	Normalt sett säkerställer `git write-tree` att objekten som refereras till av katalogen finns i objektdatabasen. Det här alternativet inaktiverar denna kontroll.

`--prefix=<prefix>/`::
	Skriver ett trädobjekt som representerar en underkatalog _<prefix>_. Detta kan användas för att skriva trädobjektet för ett delprojekt som finns i den namngivna underkatalogen.

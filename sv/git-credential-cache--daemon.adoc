git-credential-cache{litdd}daemon(1)
====================================

NAMN
----
git-credential-cache--daemon - Lagra autentiseringsuppgifter tillfälligt i minnet

SYNOPSIS
--------
[verse]
'git credential-cache{litdd}daemon' [--debug] <socket-sökväg>

BESKRIVNING
-----------

NOTE: Du vill förmodligen inte anropa det här kommandot själv; det startas automatiskt när du använder linkgit:git-credential-cache[1].

Kommandot lyssnar på Unix-domänsocketen som anges av `<socket-sökväg>` för `git-credential-cache`-klienter. Klienter kan lagra och hämta inloggningsuppgifter. Varje autentiseringsuppgift lagras under en tidsgräns som anges av klienten; när inga uppgifter längre lagras avslutas daemonen.

Om alternativet `--debug` anges stänger inte daemonen sin stderr-ström, och kan mata ut extra diagnostik till den även efter att den har börjat lyssna efter klienter.

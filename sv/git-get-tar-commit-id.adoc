git-get-tar-commit-id(1)
========================

NAMN
----
git-get-tar-commit-id - Extrahera commit-ID från ett arkiv skapat med git-archive


SYNOPSIS
--------
[verse]
'git get-tar-commit-id'


BESKRIVNING
-----------

Läs ett tar-arkiv skapat av 'git archive' från standardindata och extrahera det commit-ID som lagras i det. Det läser bara de första 1024 byten av indata, så dess körtid påverkas inte särskilt mycket av storleken på tar-arkivet.

Om inget inchecknings-ID hittas avslutas 'git get-tar-commit-id' tyst med returkoden 1. Detta kan hända om arkivet inte har skapats med 'git archive' eller om den första parametern i 'git archive' har varit ett träd-ID i stället för ett inchecknings-ID eller en tagg.

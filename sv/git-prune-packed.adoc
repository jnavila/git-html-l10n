git-prune-packed(1)
===================

NAMN
----
git-prune-packed - Ta bort extra objekt som redan finns i packfiler


SYNOPSIS
--------
[verse]
'git prune-packed' [-n | --dry-run] [-q | --quiet]


BESKRIVNING
-----------
Programmet söker i `$GIT_OBJECT_DIRECTORY` efter alla objekt som för närvarande finns i en pack-fil såväl som i de oberoende objektkatalogerna.

Alla sådana extra objekt tas bort.

Ett paket är en samling objekt, individuellt komprimerade, med deltakomprimering tillämpad, lagrade i en enda fil, med en tillhörande indexfil.

Paket används för att minska belastningen på spegelsystem, backupmotorer, disklagring etc.


ALTERNATIV
----------
-n::
--dry-run::
        Ta inte bort några objekt, visa bara de som skulle ha tagits bort.

-q::
--quiet::
	Tysta av förloppsindikatorn.

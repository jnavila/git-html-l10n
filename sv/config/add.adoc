`add.ignoreErrors`::
`add.ignore-errors` (föråldrad)::
	Anger att `git add` ska fortsätta lägga till filer när vissa filer inte kan läggas till på grund av indexeringsfel.
ifdef::git-add[]
	Motsvarande alternativet `--ignore-errors`.
endif::git-add[]
ifndef::git-add[]
	Motsvarar alternativet `--ignore-errors` i linkgit:git-add[1].
endif::git-add[]
	`add.ignore-errors` är föråldrad eftersom den inte följer den vanliga namngivningskonventionen för
	konfigurationsvariabler.

`clone.defaultRemoteName`::
	Namnet på den fjärren som ska skapas vid kloning av ett arkiv. Standardvärdet är `origin`.
ifdef::git-clone[]
	Det kan åsidosättas genom att använda 
	kommandoradsalternativet `--origin`.
endif::[]
ifndef::git-clone[]
	Den kan åsidosättas genom att skicka
	kommandoradsalternativet `--origin` till linkgit:git-clone[1].
endif::[]

`clone.rejectShallow`::
	Avvisa kloning av ett kodförråd om det är ett ytligt sådant; detta kan åsidosättas genom att skicka alternativet `--reject-shallow` på kommandoraden.
ifndef::git-clone[]
	Se linkgit:git-clone[1].
endif::[]

`clone.filterSubmodules`::
	Om ett partiellt klonfilter tillhandahålls (se `--filter` i linkgit:git-rev-list[1]) och `--recurse-submodules` används, använd även filtret på undermoduler.

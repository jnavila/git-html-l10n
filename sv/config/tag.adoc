`tag.forceSignAnnotated`::
	Ett booleskt värde som anger om skapade annoterade taggar ska vara GPG-signerade. Om `--annotate` anges på kommandoraden har det företräde framför detta alternativ.

`tag.sort`::
ifdef::git-tag[]
Denna variabel styr sorteringsordningen för taggar när de visas av `git-tag`.
endif::git-tag[]
ifndef::git-tag[]
Denna variabel styr sorteringsordningen för taggar när de visas av linkgit:git-tag[1].
endif::git-tag[]
Utan alternativet `--sort=<value>` anges, kommer värdet på denna variabel att användas som standard.

`tag.gpgSign`::
	En boolesk inställning som anger om alla taggar ska GPG-signeras. Att använda detta i automatiserade skript kan leda till att många taggar signeras. Det är därför praktiskt att använda en agent för att slippa skriva GPG-lösenfrasen flera gånger. Observera att detta alternativ inte påverkar signering som aktiverats med alternativen `-u <keyid>` eller `--local-user=<keyid>`.

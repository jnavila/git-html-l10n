git-verify-commit(1)
====================

NAMN
----
git-verify-commit - Kontrollera GPG-signaturen för incheckningar

SYNOPSIS
--------
[synopsis]
git verify-commit [-v | --verbose] [--raw] <incheckning>...

BESKRIVNING
-----------
Validerar GPG-signaturen som skapats av `git commit -S` på incheckningsobjekten som anges på kommandoraden.

ALTERNATIV
----------
`--raw`::
	Skriv ut rå GPG-statusutdata på standardfel i stället för normal människoläsbar utdata.

`-v`::
`--verbose`::
	Skriv ut innehållet i incheckningsobjektet innan du validerar det.

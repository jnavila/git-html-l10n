git-verify-commit(1)
====================

NAMN
----
git-verify-commit - Kontrollera GPG-signaturen för incheckningar

SYNOPSIS
--------
[synopsis]
git verify-commit [-v | --verbose] [--raw] <incheckning>...

BESKRIVNING
-----------
Validerar GPG-signaturen som skapats av `git commit -S` på incheckning-objekten som anges på kommandoraden.

ALTERNATIV
----------
`--raw`::
	Print the raw gpg status output to standard error instead of the normal human-readable output.

`-v`::
`--verbose`::
	Skriv ut innehållet i inchecknings-objektet innan du validerar det.

git-citool(1)
=============

NAMN
----
git-citool - Grafiskt alternativ till git-commit

SYNOPSIS
--------
[verse]
'git citool'

BESKRIVNING
-----------
Ett Tcl/Tk-baserat grafiskt gränssnitt för att granska modifierade filer, köa dem i indexet, ange ett incheckning-meddelande och registrera den nya incheckning på den aktuella grenen. Detta gränssnitt är ett alternativ till det mindre interaktiva programmet 'git commit'.

''git citool'' är egentligen ett standardalias för `'git gui citool''. Se linkgit:git-gui[1] för mer information.

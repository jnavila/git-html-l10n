git-sh-i18n{litdd}envsubst(1)
=============================

NAMN
----
git-sh-i18n--envsubst - Gits egen envsubst(1) för i18n-reservfunktioner

SYNOPSIS
--------
[verse]
eval_gettext () {
	printf "%s" "$1" | (
		export PATH $('git sh-i18n{litdd}envsubst' --variables "$1");
		'git sh-i18n{litdd}envsubst' "$1"
	)
}

BESKRIVNING
-----------

Det är inte ett kommando som slutanvändaren någonsin skulle vilja köra. Dokumentationen är avsedd för personer som studerar lågnivåskript och/eller skriver nya.

'git sh-i18n{litdd}envsubst' är Gits nedskalade kopia av GNU-programmet `envsubst(1)` som följer med GNU gettext-paketet. Det används internt av linkgit:git-sh-i18n[1] för att interpolera variablerna som skickas till funktionen `eval_gettext`.

Inga löften ges om gränssnittet, eller att det här programmet inte kommer att försvinna utan förvarning i nästa version av Git. Använd det inte.

git-merge-one-file(1)
=====================

NAMN
----
git-merge-one-file - Standardhjälpprogrammet att använda med git-merge-index


SYNOPSIS
--------
[verse]
'git merge-one-file'

BESKRIVNING
-----------
Det här är standardhjälpprogrammet som används med 'git merge-index' för att lösa en sammanslagning efter den triviala sammanslagningen som gjorts med 'git read-tree -m'.

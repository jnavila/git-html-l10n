`add.ignoreErrors`::
`add.ignore-errors` (非推奨)::
	インデックス作成エラーにより一部のファイルを追加できない場合でも、`git add` がファイルの追加を続けるように指示します。
ifdef::git-add[]
	`--ignore-errors` オプションと同等です。
endif::git-add[]
ifndef::git-add[]
	linkgit:git-add[1] の `--ignore-errors` オプションと同等です。
endif::git-add[]
	`add.ignore-errors` は、設定変数の通常の命名規則に従っていないため
	非推奨です。

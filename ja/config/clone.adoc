`clone.defaultRemoteName`::
	リポジトリのクローン作成時に作成するリモートの名前。デフォルトは `origin` です。
ifdef::git-clone[]
	`--origin` コマンドラインオプションを渡すことで上書き
	できます。
endif::[]
ifndef::git-clone[]
	linkgit:git-clone[1] に `--origin` コマンドラインオプションを
	渡すことで上書きできます。
endif::[]

`clone.rejectShallow`::
	リポジトリが浅いクローンの場合、クローン作成を拒否します。これはコマンドラインで `--reject-shallow` オプションを渡すことで上書きできます。
ifndef::git-clone[]
	linkgit:git-clone[1]を参照してください。
endif::[]

`clone.filterSubmodules`::
	部分クローンフィルターが提供され（linkgit:git-rev-list[1] の `--filter` を参照）、`--recurse-submodules` が使用されている場合、サブモジュールにもフィルターを適用します。

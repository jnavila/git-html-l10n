`init.templateDir`::
	テンプレートがコピーされるディレクトリを指定します。{see-git-init}
`init.defaultBranch`::
	新しいリポジトリを初期化するときなどに、デフォルトのブランチ名を上書きすることができます。
`init.defaultObjectFormat`::
	新しいリポジトリのデフォルトのオブジェクトフォーマットを上書きすることができます。linkgit:git-init[1] の `--object-format=` を参照してください。コマンドラインオプションと `GIT_DEFAULT_HASH` 環境変数の両方がこの設定よりも優先されます。
`init.defaultRefFormat`::
	新しいリポジトリのデフォルトの参照ストレージフォーマットを上書きすることができます。linkgit:git-init[1] の `--ref-format=` を参照してください。コマンドラインオプションと `GIT_DEFAULT_REF_FORMAT` 環境変数の両方がこの設定よりも優先されます。

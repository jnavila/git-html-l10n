`clone.defaultRemoteName`::
	El nombre del remoto a crear cuando se clona un repositorio. Predeterminado a `origin`.
ifdef::git-clone[]
	Puede anularse pasando la opción de línea de comando
	 `--origin`.
endif::[]
ifndef::git-clone[]
	Puede anularse pasando a linkgit:git-clone[1] 
	la opción de línea de comandos `--origin`.
endif::[]

`clone.rejectShallow`::
	Rechaza la clonación de un repositorio si es uno superficial; esto puede anularse pasando la opción `--reject-shallow` a la línea de comandos.
ifndef::git-clone[]
	Ver linkgit:git-clone[1].
endif::[]

`clone.filterSubmodules`::
	Si se proporciona un filtro de clonación parcial (ver `--filter` en linkgit:git-rev-list[1]) y se usa`--recurse-submodules`, también aplica el filtro a submódulos.

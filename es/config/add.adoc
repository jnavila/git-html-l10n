`add.ignoreErrors`::
`add.ignore-errors` (obsoleto)::
	Le indica a `git add` que continúe agregando ficheros cuando alguno no puede ser agregado por errores de indexación.
ifdef::git-add[]
	Equivalente a la opción `--ignore-errors`.
endif::git-add[]
ifndef::git-add[]
	Equivalente a la opción `--ignore-errors` de linkgit:git-add[1].
endif::git-add[]
	`add.ignore-errors` es obsoleto, ya que no sigue la acostumbrada
	convención de nombres para variables de configuración.

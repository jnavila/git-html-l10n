git-diff-files(1)
=================

НАЗВА
-----
git-diff-files - Порівнює файли в робочому дереві та індексі


СИНОПСИС
--------
[verse]
'git diff-files' [-q] [-0 | -1 | -2 | -3 | -c | --cc] [<common-diff-options>] [<path>...]

ОПИС
----
Порівнює файли в робочому дереві та індексі. Якщо шляхи вказані, порівнюються лише ті, що мають іменовані шляхи. В іншому випадку порівнюються всі записи в індексі. Формат виводу такий самий, як для 'git diff-index' та 'git diff-tree'.

ОПЦІЇ
-----
include::diff-options.adoc[]

-1 --base::
-2 --ours::
-3 --theirs::
-0::
	Різниця з "базовою" версією, "нашою гілкою" або "їхньою гілкою" відповідно. З цими опціями різниці для об'єднаних записів не відображаються.
+
За замовчуванням виконується порівняння з нашою гілкою (-2) та чисто вирішеними шляхами. Можна задати опцію -0, щоб пропустити вивід порівняння для необ'єднаних записів і просто показати "Не об'єднано".

-c::
--cc::
	Це порівнює етап 2 (наша гілка), етап 3 (їхня гілка) та робочий файл дерева, і виводить об'єднану різницю, подібно до того, як 'diff-tree' показує коміт злиття з цими прапорцями.

-q::
	Зберігайте тишу навіть для неіснуючих файлів

git-update-server-info(1)
=========================

НАЗВА
-----
git-update-server-info — Оновлення допоміжного інформаційного файлу для підтримки простих серверів


СИНОПСИС
--------
[verse]
'git update-server-info' [-f | --force]

ОПИС
----
Неавторизований сервер, який не генерує пакунки на льоту, повинен мати деякі допоміжні інформаційні файли в теках $GIT_DIR/info та $GIT_OBJECT_DIRECTORY/info, щоб допомогти клієнтам дізнатися, які посилання та пакунки має сервер. Ця команда генерує такі допоміжні файли.

ОПЦІЇ
-----
-f::
--force::
	Оновіть інформаційні файли з нуля.

ВИВІД
-----

Наразі команда оновлює такі файли. Опис їх призначення дивіться у linkgit:gitrepository-layout[5]:

* objects/info/packs

* info/refs

git-verify-tag(1)
=================

НАЗВА
-----
git-verify-tag — Перевірка GPG-підпису тегів

СИНОПСИС
--------
[synopsis]
git verify-tag [-v | --verbose] [--format=<format>] [--raw] <tag>...

ОПИС
----
Перевіряє підпис gpg, створений командою `git tag`, у обʼєктах тегів, перелічених у командному рядку.

ОПЦІЇ
-----
`--raw`::
	Виводить необроблений вивід стану gpg у стандартний потік помилок замість звичайного виводу, зручного для людини.

`-v`::
`--verbose`::
	Виводить вміст обʼєкта тегу перед його перевіркою.

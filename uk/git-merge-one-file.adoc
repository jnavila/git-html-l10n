git-merge-one-file(1)
=====================

НАЗВА
-----
git-merge-one-file — Стандартна допоміжна програма для використання з git-merge-index


СИНОПСИС
--------
[verse]
'git merge-one-file'

ОПИС
----
Це стандартна допоміжна програма для використання з 'git merge-index' для завершення злиття після того як проста частина злиття вже виконанна за допомогою 'git read-tree -m'.

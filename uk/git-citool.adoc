git-citool(1)
=============

НАЗВА
-----
git-citool — Графічна альтернатива git-commit

СИНОПСИС
--------
[verse]
'git citool'

ОПИС
----
Графічний інтерфейс на базі Tcl/Tk, призначений для перегляду змінених файлів, додавання їх до індексу, введення повідомлення про коміт та запису нового коміту в поточну гілку. Цей інтерфейс є альтернативою менш інтерактивній програмі 'git commit'.

'git citool' насправді є стандартним псевдонімом для `git gui citool`. Див. linkgit:git-gui[1] для отримання додаткової інформації.

`add.ignoreErrors`::
`add.ignore-errors` (застарілий)::
	Наказує `git add` продовжувати додавання файлів, якщо деякі файли неможливо додати через помилки індексації.
ifdef::git-add[]
	Еквівалентно опції `--ignore-errors`.
endif::git-add[]
ifndef::git-add[]
	Еквівалентно опції `--ignore-errors` у linkgit:git-add[1].
endif::git-add[]
	`add.ignore-errors` застарів, оскільки він не відповідає звичайним
	правила іменування для змінних конфігурації.

sequence.editor::
	Текстовий редактор, який використовується командою `git rebase -i` для редагування файлу інструкцій перебазування. Це значення призначене для інтерпретації оболонкою під час використання. Його можна замінити за допомогою змінної середовища `GIT_SEQUENCE_EDITOR`. Якщо це значення не вказано, замість нього використовується стандартний редактор повідомлень комітів.

git-unpack-file(1)
==================

НАЗВА
-----
git-unpack-file — Створює тимчасовий файл із вмістом блобу



СИНОПСИС
--------
[verse]
'git unpack-file' <blob>

ОПИС
----
Створює файл, що містить вміст блобу, вказаного за допомогою sha1. Ця функція
повертає імʼя тимчасового файлу у такому форматі:
	.merge_file_XXXXX

ОПЦІЇ
-----
<blob>::
	Має бути ідентифікатором блобу

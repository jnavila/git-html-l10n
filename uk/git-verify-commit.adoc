git-verify-commit(1)
====================

НАЗВА
-----
git-verify-commit — Перевірка GPG-підпису комітів

СИНОПСИС
--------
[synopsis]
git verify-commit [-v | --verbose] [--raw] <commit>...

ОПИС
----
Перевіряє підпис GPG, створений командою `git commit -S`, у обʼєктів комітах, заданих у командному рядку.

ОПЦІЇ
-----
`--raw`::
	Виводить необроблений вивід стану gpg у стандартний потік помилок замість звичайного виводу, зручного для людини.

`-v`::
`--verbose`::
	Виводить вміст обʼєкта commit перед його перевіркою.

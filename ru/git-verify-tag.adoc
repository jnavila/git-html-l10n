git-verify-tag(1)
=================

НАЗВАНИЕ
--------
git-verify-tag — проверка GPG-подписи меток

ОБЗОР
-----
[synopsis]
'git verify-tag' [-v | --verbose] [--format=<формат>] [--raw] <метка>...

ОПИСАНИЕ
--------
Проверяет gpg-подпись, созданную `git tag` в объектах меток, перечисленных в командной строке.

ПАРАМЕТРЫ
---------
`--raw`::
	Выводить необработанный вывод статуса gpg в стандартный вывод ошибок вместо обычного человекочитаемого вывода.

`-v`::
`--verbose`::
	Вывести содержимое объекта метки перед его проверкой.

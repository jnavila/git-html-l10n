`-U<n>`::
`--unified=<n>`::
	Создать списки изменений с _<n>_ строками контекста. По умолчанию `diff.context` или 3, если переменная конфигурации не задана.

`--inter-hunk-context=<n>`::
	Выводить в качестве контекста между блоками изменений до _<n>_ строк, тем самым объединяя близкие блоки изменений. По умолчанию равно значению переменной конфигурации `diff.interHunkContext` или 0, если она не установлена.

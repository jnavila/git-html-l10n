git-merge-one-file(1)
=====================

НАЗВАНИЕ
--------
git-merge-one-file - Стандартная программа-помощник для использования совместно с git-merge-index


ОБЗОР
-----
[verse]
'git merge-one-file'

ОПИСАНИЕ
--------
Это стандартная программа-помощник, используемая совместно с 'git-merge-index', чтобы завершить слияние после того как тривиальная часть слияния уже выполнена 'git read-tree -m'.

GIT
---
Является частью пакета linkgit:git[1]

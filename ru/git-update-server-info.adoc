git-update-server-info(1)
=========================

НАЗВАНИЕ
--------
git-update-server-info — обновление вспомогательного информационного файла для помощи примитивным серверам


ОБЗОР
-----
[verse]
'git update-server-info' [-f | --force]

ОПИСАНИЕ
--------
Примитивный сервер, который не выполняет генерацию пакетов на лету, должен иметь некоторые вспомогательные информационные файлы в каталогах $GIT_DIR/info и $GIT_OBJECT_DIRECTORY/info, чтобы помочь клиентам обнаружить, какие ссылки и пакеты есть на сервере. Эта команда создаёт такие вспомогательные файлы.

ПАРАМЕТРЫ
---------
-f::
--force::
	Обновить информационные файлы с нуля.

ВЫВОД
-----

В настоящее время команда обновляет следующие файлы. Пожалуйста, обратитесь к linkgit:gitrepository-layout[5] для описания их назначения:

* objects/info/packs

* info/refs

GIT
---
Является частью пакета linkgit:git[1]

git-sh-i18n{litdd}envsubst(1)
=============================

НАЗВАНИЕ
--------
git-sh-i18n--envsubst — Собственный envsubst(1) Git для запасных вариантов i18n

ОБЗОР
-----
[verse]
eval_gettext () {
	printf "%s" "$1" | (
		export PATH $('git sh-i18n{litdd}envsubst' --variables "$1");
		'git sh-i18n{litdd}envsubst' "$1"
	)
}

ОПИСАНИЕ
--------

Это не та команда, которую конечный пользователь захотел бы запускать. Никогда. Эта документация предназначена для людей, изучающих внутренние сценарии (plumbing scripts) и/или пишущих новые.

'git sh-i18n{litdd}envsubst' — это урезанная копия программы GNU `envsubst(1)`, поставляемой с пакетом GNU gettext, принадлежащая Git. Она используется внутренне linkgit:git-sh-i18n[1] для интерполяции переменных, передаваемых функции `eval_gettext`.

Никаких обещаний относительно интерфейса или того, что эта программа не исчезнет без предупреждения в следующей версии Git, не даётся. Не используйте её.

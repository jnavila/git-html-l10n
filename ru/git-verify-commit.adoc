git-verify-commit(1)
====================

НАЗВАНИЕ
--------
git-verify-commit — проверка GPG-подписи коммитов

ОБЗОР
-----
[synopsis]
'git verify-commit' [-v | --verbose] [--raw] <коммит>...

ОПИСАНИЕ
--------
Проверяет GPG-подпись, созданную `git commit -S`, для объектов коммитов, указанных в командной строке.

ПАРАМЕТРЫ
---------
`--raw`::
	Выводить необработанный вывод статуса gpg в стандартный вывод ошибок вместо обычного человекочитаемого вывода.

`-v`::
`--verbose`::
	Вывести содержимое объекта коммита перед его проверкой.

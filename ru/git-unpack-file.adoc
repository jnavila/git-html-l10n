git-unpack-file(1)
==================

НАЗВАНИЕ
--------
git-unpack-file — создание временного файла с содержимым blob-объекта



ОБЗОР
-----
[verse]
'git unpack-file' <blob-объект>

ОПИСАНИЕ
--------
Создаёт файл, содержащий содержимое blob-объекта, указанного sha1. Он
возвращает имя временного файла в следующем формате:
	.merge_file_XXXXX

ПАРАМЕТРЫ
---------
<blob-объект>::
	Должен быть идентификатором blob-объекта

GIT
---
Является частью пакета linkgit:git[1]

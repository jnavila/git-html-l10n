`clone.defaultRemoteName`::
	Имя, которое будет использоваться для удалённого репозитория при его клонировании. По умолчанию: `origin`.
ifdef::git-clone[]
	Его можно переопределить, передав параметр командной строки
	`--origin`.
endif::[]
ifndef::git-clone[]
	Его можно переопределить, передав linkgit:git-clone[1]
	параметр командной строки `--origin`.
endif::[]

`clone.rejectShallow`::
	Отказаться от клонирования репозитория, если он является частичным (shallow); это поведение можно переопределить, передав параметр командной строки `--reject-shallow`.
ifndef::git-clone[]
	См. linkgit:git-clone[1].
endif::[]

`clone.filterSubmodules`::
	Если задан фильтр частичного клонирования (см. `--filter` в linkgit:git-rev-list[1]) и используется `--recurse-submodules`, то применять фильтр также и к подмодулям.

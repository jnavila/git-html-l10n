`add.ignoreErrors`::
`add.ignore-errors` (устаревший)::
	Даёт указание `git.add` продолжить добавление файлов в ситуации, когда не удалось добавить некоторые файлы из-за ошибок индексирования.
ifdef::git-add[]
	Эквивалентно параметру `--ignore-errors`.
endif::git-add[]
ifndef::git-add[]
	Эквивалентно параметру `--ignore-errors` linkgit:git-add[1].
endif::git-add[]
	`add.ignore-errors` считается устаревшим (deprecated), так как он не следует
	устоявшимся конвенциям наименования переменных конфигурации.

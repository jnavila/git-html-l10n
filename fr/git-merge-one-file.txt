git-merge-one-file(1)
=====================

NOM
---
git-merge-one-file - Ce programme auxiliaire standard à utiliser avec git-merge-index


SYNOPSIS
--------
[verse]
'git-merge-one-file'

DESCRIPTION
-----------
C'est le programme d'aide standard à utiliser avec 'git merge-index' pour résoudre une fusion après la fusion triviale réalisée avec 'git read-tree -m'.

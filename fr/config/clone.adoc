`clone.defaultRemoteName`::
	Le nom du distant à créer lors du clonage d'un dépôt. Par défaut `origin`.
ifdef::git-clone[]
	Il peut être remplacé par l'option de ligne de commande 
	`--origin`.
endif::[]
ifndef::git-clone[]
	Il peut être surchargé en passant l'option de ligne de commande
	`--origin` à linkgit:git-clone[1].
endif::[]

`clone.rejectShallow`::
	Rejeter le clonage d'un dépôt s'il s'agit d'un dépôt superficiel ; cela peut être surchargé en passant l'option `--reject-shallow` sur la ligne de commande.
ifndef::git-clone[]
	Voir linkgit:git-clone[1]
endif::[]

`clone.filterSubmodules`::
	Si un filtre de clone partiel est fourni (voir `--filter` dans linkgit:git-rev-list[1]) et `--recurse-submodules` est utilisé, appliquer également le filtre aux sous-modules.

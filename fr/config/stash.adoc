`stash.index`::
	Si la valeur est true, `git stash apply` et `git stash pop` se comportera comme si `--index` était fourni. Par défaut à false.
ifndef::git-stash[]
Voir les descriptions dans linkgit:git-stash[1].
+
Cela affecte également les invocations de linkgit:git-stash[1] via `--autostash` des commandes telles que linkgit:git-merge[1], linkgit:git-rebase[1] et linkgit:git-pull[1].
endif::git-stash[]

`stash.showIncludeUntracked`::
	Si c'est réglé à `true`, la commande `git stash show` montrera les fichiers non suivis d'une entrée de remisage. Vaut par défaut `false`. {see-show}

`stash.showPatch`::
	Si cela est vrai, la commande `git stash show` sans option affichera l'entrée de remisage sous forme de rustine. Par défaut à faux. {see-show}

`stash.showStat`::
	Si cela est à true, la commande `git stash show` sans option affichera un diffstat de l`entrée de remisage. Par défaut à true. {see-show}

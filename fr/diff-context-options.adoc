`-U<n>`::
`--unified=<n>`::
	Générer des diffs avec _<n>_ lignes de contexte. Le nombre de lignes de contexte par défaut à `diff.context` ou 3 si la variable de configuration est indéfinie. (`-U` sans `<n>` est silencieusement accepté comme synonyme de `-p` en raison d'un accident historique.

`--inter-hunk-context=<n>`::
	Afficher le contexte entre des sections de diff, jusqu'au _<nombre>_ spécifié de lignes, fusionnant de ce fait les sections qui sont proches. Par défaut, `diff.interHunkContext` ou 0 si l'option de configuration n'est pas configurée.

git-annotate(1)
===============

NOM
---
git-annotate - Annote les lignes d'un fichier avec les informations de
commit

SYNOPSIS
--------
[verse]
'git annotate' [<options>] <fichier> [<révision>]

DESCRIPTION
-----------
Annote chaque ligne dans le fichier donné avec les informations du commit
qui a introduit la ligne. Annote optionnellement à partir d'une révision
donnée.

The only difference between this command and linkgit:git-blame[1] is that
they use slightly different output formats, and this command exists only for
backward compatibility to support existing scripts, and provide a more
familiar command name for people coming from other SCM systems.

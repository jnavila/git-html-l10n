git-init-db(1)
==============

NAME
----
git-init-db - Creates an empty Git repository


ÜBERSICHT
---------
[verse]
'git init-db' [-q | --quiet] [--bare] [--template=<Template_Verzeichnis>] [--separate-git-dir <Git-Verzeichnis>] [--shared[=<Zugriffsrechte>]]


BESCHREIBUNG
------------

Dies ist ein Synonym für linkgit:git-init[1]. Bitte lesen Sie die Dokumentation dieses Kommandos.

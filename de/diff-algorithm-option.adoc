`--diff-algorithm=(patience|minimal|histogram|myers)`::
	Einen Diff-Algorithmus auswählen. Die verschiedenen Alternativen folgen unten:
+
--
   `default`;;
   `myers`;;
	Der allgemeine „gefräßige“ Diff-Algorithmus. Derzeit ist dies die Standardeinstellung.
   `minimal`;;
	Verwendet zusätzliche Zeit darauf, um das kleinstmögliche Diff zu erzeugen.
   `patience`;;
	Verwenden Sie den "patience diff"-Algorithmus zum Generieren von Patches.
   `histogram`;;
	Dieser Algorithmus erweitert den 'patience'-Algorithmus, um „allgemeine Elemente mit geringer Häufigkeit zu berücksichtigen“.
--
+
Wenn Sie die Variable `diff.algorithm` auf einen nicht voreingestellten Wert konfiguriert haben und den voreingestellten Wert verwenden wollen, dann müssen Sie die Option `--diff-algorithm=default` verwenden.
